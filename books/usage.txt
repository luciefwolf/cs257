BookSearch - this is a program that displays information about some books 

SYNOPSIS: 
python3 BookSearch.py searchAuthor "key phrase" [OPTION]
python3 BookSearch.py searchTitle "key phrase" [OPTION]
python3 BookSearch.py range startDate endDate [OPTION]
python3 BookSearch.py help

OPTIONS: 
-t, --title: sort by book titles alphabetically
-y, --year: sort by publication year
-n, --author: sort by author's last name
